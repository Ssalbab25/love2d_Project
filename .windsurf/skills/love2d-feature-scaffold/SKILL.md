# 목적

LÖVE2D에서 신규 게임플레이 기능을 구현하거나 기존 기능을 리팩터링할 때 이 스킬을 사용합니다.

## 절차

1. 기능이 영향을 주는 파일과 게임 상태를 식별합니다.
2. 현재 동작을 보존하는 최소 구조 변경안을 제시합니다.
3. update와 draw 경로를 책임 경계가 보이도록 구현합니다.
4. 간결한 사용 방법과 검증 메모를 추가합니다.

## 출력 형식

- 변경 파일
- 동작 변화
- 검증 단계
