# /release-check

**설명:** 이 LÖVE2D 및 Lua 워크스페이스에서 배포 또는 릴리스 전 품질 점검과 통합 성능 진단을 수행합니다.

## 실행 절차 및 프로세스

1. **상태 변경 검토:** 마지막 릴리스 커밋 이후 현재 Git 작업 트리에서 수정, 추가 또는 삭제된 모든 파일 목록을 취합합니다.
2. **런타임 및 시뮬레이션 위험 평가:**
   - 메인 루프(`update/draw`)에 성능 급등이나 빈번한 임시 객체 생성이 들어갔는지 확인합니다.
   - 전역 스코프(`_G`) 오염 시도가 존재하는지 검출합니다.
3. **코드 일관성 스캔:**
   - StyLua 및 Luacheck 설정 준수 여부를 확인합니다.
   - 모듈화 패턴이 `local M = {} ... return M` 구조를 충실히 만족하는지 점검합니다.
4. **품질 검사 및 자가 진단 리포트:**
   - 발견된 잠재적 버그와 리스크 요소를 등급별(상/중/하)로 표시합니다.
   - 원활한 배포를 위한 긴급 릴리스 체크리스트와 검증 명령어를 최종 출력합니다.
